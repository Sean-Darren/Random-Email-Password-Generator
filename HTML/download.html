Email and Password Generator

Extension

Step 1
Download the Extension Files

Click the download link to get the extension's ZIP file. Extract the ZIP file to a folder on your computer.

Download
Step 2
Enable Developer Mode in Chrome

Open Google Chrome. Search chrome://extensions/ in the address bar. Turn on the Developer Mode

Step 3
Load the Extensions

Click the "Load unpacked" button. Select the folder where you extracted the extension files.

Step 4
Set Up the Backend Server

1. Install Python

2. Open Terminal and navigate to the extension folder: 

cd /path/to/your/extension/folder

3. Install Flask:

python3 -m pip install flask

4. The server will run at:

http://127.0.0.1:5000
Step 5
Use the Extension

Click the extension icon in Chrome. Click "Generate Email" or "Generate Password" — the generated details will autofill any email or password fields on the page.

Notes: Ensure the Flask server is running in the background for the extension to work.
